# SafeArea 安全区
顶部或者底部安全区域适配，防止内容被遮挡

## 基本用法
```html
<wd-safe-area position="top" />
<wd-safe-area position="bottom" />
```

## Attributes

| 参数             | 说明                                  | 类型            | 可选值 | 默认值 | 最低版本 |
| ---------------- | ------------------------------------- | --------------- | ------ | ------ | -------- |
| position      | 安全区的位置                        | string          | top \| bottom | - | -        |

## 外部样式类

| 类名         | 说明       | 最低版本 |
| ------------ | ---------- | -------- |
| custom-class | 根节点样式 | -        |
