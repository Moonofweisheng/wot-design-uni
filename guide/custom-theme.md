# 自定义主题

Wot UI 每个组件基本都有自定义类名 custom-class，可以在组件根节点加入你页面上的类名，进行样式修改。

### 中性色

中性色用于文本、背景和边框颜色。通过运用不同的中性色，来表现层次结构。
