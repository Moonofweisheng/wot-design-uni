## ✨ 特性

- 🚀 支持 APP、H5、微信小程序 等平台.
- 🚀 50+ 个高质量组件，覆盖移动端主流场景.
- 💪 使用 Typescript 构建，提供良好的组件类型系统.
- 💪 采用 Vue3 最新特性，提升组件性能.
- 📖 提供丰富的文档和组件示例.
- 🎨 支持主题定制，可以定制 scss 变量以及组件的样式自定义.
- 🍭 支持暗黑模式
